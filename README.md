# Daikibo Machine Downtime Analysis

## 📌 Project Overview
This project analyzes machine telemetry data and employee equality data to identify operational inefficiencies and workplace disparities across global factories.

## 🎯 Objectives
- Identify which factory experienced the most machine downtime  
- Determine which machine types contributed most to failures  
- Classify gender equality scores to highlight potential disparities  

## 🛠 Tools Used
- Tableau (Dashboard & Visualization)
- Excel (Data Classification)
- JSON (Telemetry Data)

## 📊 Key Insights
- Daikibo Factory Seiko recorded the highest machine downtime  
- Laser Welder and Laser Cutter machines contributed most to failures  
- Higher inequality scores were observed in senior roles across certain locations
